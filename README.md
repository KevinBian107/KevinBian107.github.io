# kbian.org

Personal academic website for **Kaiwen Bian** — 4th-year undergraduate at UC San Diego studying Data Science and Cognitive Behavioral Neuroscience.

Live at **[kbian.org](https://kbian.org)**

## Research Areas

- **Embodied Agents & RL** — neuromechanical emulation, motor control, adversarial multi-agent training
- **Representation & Topology** — topological data analysis, latent space geometry, dimensionality reduction
- **Generative Models** — molecular graph generation, deep state space models for biomechanics
- **Language Models & Multi-Agent** — multi-agent collaboration, multimodal embodied simulation
- **Computational Neuroscience** — virtual rodent models, neural population structure, stress/rumination analysis

## Structure

```
index.html              Home page (profile + about)
research.html           Full project listing with category filters
research/               Research detail pages
projects/               Project detail pages
stylesheets/styles.css  All site styling
javascripts/            Translate widget, viewport fix, category filter
assets/                 Images, GIFs, PDFs
```

## Development

Static HTML/CSS/JS — no build step. Edit files directly and push to `main` for GitHub Pages deployment.
