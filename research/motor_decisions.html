Emergent Motor Programs from Discrete Bottlenecks in Whole-Body Biomechanical Control

📅 Project Timeline: Jun 2025 - Now

Reinforcement learning can produce high-fidelity whole-body control for detailed biomechanical models, but the resulting policies are continuous black boxes offering no intermediate structure that can be inspected or reused. In this work, we train a neural policy to imitate natural rodent behavior in a physically simulated 37-joint body, but force all control through a vector-quantized bottleneck: the policy must select one of \(K\) discrete codes, hold it for several hundred milliseconds, and a decoder translates that code into joint torques given current proprioception. The codes emerge end-to-end from task pressure — they are not hand-designed.

Early results reveal that individual codes exhibit superposition: the same code produces recognizably consistent behavior across body configurations, yet proprioceptive context gates execution. Temporal code sequences are highly structured and capturable by a simple Hidden Markov Model, suggesting that the discrete bottleneck creates a programmable motor interface that decouples what to do from how to move. This project builds on earlier work on dynamic modeling for biomechanical planning. See MimicDyn: Dynamic Model for Planning & Generating Biomechanically Realistic Behaviors with Deep State Space Methods for the previous iteration that motivated this direction.
