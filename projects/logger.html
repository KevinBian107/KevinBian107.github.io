log(ger): Personal Productivity Tracking Web App

📅 Project Timeline: Feb 2026 - Now

Visit the new Logger website that we have created.

A ground-up rebuild of my earlier Python/Jupyter study database into a modern full-stack web application. log(ger) tracks study and work time across academic sessions with real-time timers, analytics dashboards, AI-powered chat, and project progression visualization — because every minute is worth logging.

log(ger) — personal productivity tracking.

GitHub Repository

log(ger) evolved from an earlier project — a personalized numerical and text database built in Python/Jupyter for tracking study data at UCSD. That original project examined different study habits through vectorized data storage and analysis workflows, and served as the foundation for this full-stack rebuild.
