Engram: Handwritten Notes to Structured Knowledge Base

📅 Project Timeline: 2026 - Now
Code (closed repo)

Engram is a personal knowledge base that transcribes handwritten notes into structured Markdown and organizes the extracted information into a navigable, interlinked repository. Named after the neural engram — the physical trace a memory leaves behind — the project aims to turn scattered ink on paper into a durable, searchable second brain.

Engram — from ink to structure, from notes to memory.
